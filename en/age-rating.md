# Age Rating

## Rating: 4+ (Suitable for all ages)

CallPlan is a productivity app for professional call preparation and is designed for use by all age groups.

---

| Category | Status |
|---|---|
| Violence or graphic content | None |

| Alcohol, tobacco, or drug references | None |
| Gambling | None |
| Horror or frightening content | None |
| Profanity or crude language | None |
| Contact with unknown individuals | None |
| In-app purchases | Optional (Subscription) |

---

### App Features

CallPlan enables:

- **Call scheduling** – Create appointments, set reminders
- **Hands-free notes** – Dictate via Siri or CarPlay
- **Automatic note delivery** – Email or messenger to known contacts
- **Calendar integration** – Sync with the system calendar

All communication features are directed exclusively at contacts selected by the user. There is no interaction with unknown individuals.

---

### Privacy & Security

CallPlan does not store user data in the cloud and does not share data with third parties. The app works fully offline. Further information: [Privacy Policy →](/en/privacy/)
